Manus Integration

To integrate with Manus AI, configure an MCP server or use the HTTP API directly.

# Example: Manus reading your chats
GET /api/v1/messages/chats
Header: X-API-Key: your_api_key

# Example: Manus searching messages
POST /api/v1/messages/search
Body: {"query": "date with Masoud", "contact": "Masoud Nayebi"}

# Example: Manus replying as assistant
POST /api/v1/assistant/reply-as-assistant
Body: {"phone": "989...", "message": "Hi, Alireza will contact you soon."}
